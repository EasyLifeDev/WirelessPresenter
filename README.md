Wireless Presenter PC
=====================

Wireless Presenter PC is the desktop companion app required by the Wireless Presenter mobile app.
It shows a QR code that lets your phone connect to your computer on the same local network.


How to use
----------

1. Install or open Wireless Presenter PC on your computer.
2. Connect your phone and computer to the same Wi-Fi network.
   If there is no Wi-Fi, enable your phone hotspot and connect the computer to it.
3. Open the Wireless Presenter mobile app.
4. Tap "Scan PC QR Code" and scan the QR code shown by the PC app.
5. After pairing, use your phone as a presentation remote, touchpad, Air Mouse or laser pointer.


Windows
-------

1. Download the Windows ZIP package.
2. Extract the ZIP file.
3. Open the extracted folder.
4. Double-click "Wireless Presenter PC" or the included executable file.

If Windows shows a security warning, choose "More info" then "Run anyway" only if you downloaded the app from the official Wireless Presenter release page.


macOS
-----

1. Download the macOS ZIP package.
2. Extract the ZIP file.
3. Open Wireless Presenter PC.

If macOS blocks the app because it was downloaded from the internet:

1. Open System Settings.
2. Go to Privacy & Security.
3. Allow Wireless Presenter PC to open.


Linux
-----

You can use one of these packages:

- AppImage: make it executable, then run it.
- DEB package: install it on Debian/Ubuntu-based systems.
- ZIP package: extract it and run the included executable or run.sh script.

AppImage example:

chmod +x wireless-presenter-desktop-linux.AppImage
./wireless-presenter-desktop-linux.AppImage

DEB example:

sudo apt install ./wireless-presenter-desktop.deb


Uninstall
---------

Windows:

- If you used the ZIP package, close Wireless Presenter PC and delete the extracted folder.
- If you created a shortcut, delete the shortcut too.

macOS:

- Close Wireless Presenter PC.
- Move the app to the Trash.
- Empty the Trash if you want to remove it completely.

Linux:

- If you used the AppImage, close the app and delete the AppImage file.
- If you used the ZIP package, close the app and delete the extracted folder.
- If you installed the DEB package, run:

sudo apt remove wireless-presenter-desktop


Network notes
-------------

- The phone and computer must be on the same local network.
- Some public, school, hotel or company Wi-Fi networks block device-to-device connections.
- If pairing does not work, use your phone hotspot and connect the computer to it.
- Make sure your firewall allows Wireless Presenter PC to accept local network connections.


Troubleshooting
---------------

If the QR code does not connect:

1. Restart Wireless Presenter PC.
2. Restart the mobile app.
3. Check that both devices are on the same Wi-Fi or hotspot.
4. Disable VPN temporarily on the phone and computer.
5. Allow the PC app through the firewall.
